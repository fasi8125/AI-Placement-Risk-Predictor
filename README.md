# 🚀 AI Placement Risk & Career Intelligence System

## 📌 Overview
An AI-powered system designed to predict student employability and support education loan decision-making.

This platform estimates:
- 🎯 Placement probability (3/6/12 months)
- 💰 Expected starting salary
- ⚠️ Placement risk level (Low / Medium / High)

It also includes a **Resume Intelligence Engine** that enhances predictions by extracting real signals from student resumes.

---

## 🧠 Problem Statement
This project addresses:

**“Linking Study-Abroad Education Loan to Career Success Using AI”**

Lenders currently lack visibility into:
- Student employability
- Placement timelines
- Salary potential

This leads to:
- Poor risk assessment
- Higher loan defaults

👉 Our solution acts as a **pre-underwriting intelligence layer**.
